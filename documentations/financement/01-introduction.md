## Présentation générale

Le module **Financement** de Papaours permet de gérer l'ensemble du processus de financement des contrats d'apprentissage. Il assure le suivi des montants, des financeurs et de la répartition des coûts de formation.

### Objectifs du module

- **Centraliser** les informations de financement liées à chaque contrat
- **Répartir** les coûts entre les différents financeurs (OPCO, employeur, etc.)
- **Suivre** l'état d'avancement du financement (reste à ventiler, taux de couverture)
- **Appliquer** les règles métier spécifiques (participation obligatoire employeur, remises)

### Périmètre fonctionnel

Le module Financement couvre :

| Fonctionnalité | Description |
|----------------|-------------|
| Dossier de financement | Création automatique à partir d'un contrat validé |
| Gestion des financeurs | OPCO, Employeur, Région, Autre |
| Prestations vendues | Pédagogie, hébergement, restauration, équipement... |
| Ventilation | Répartition des montants entre financeurs |
| Remises | Remises sur ventilation et remises globales par financeur |
| Récapitulatif | Suivi des totaux et du reste à financer |

### Prérequis

Pour utiliser le module Financement, les éléments suivants doivent être en place :

1. Un **contrat d'apprentissage** validé
2. Un **employeur** avec son OPCO rattaché (si applicable)
3. Les **prestations** configurées dans le catalogue
