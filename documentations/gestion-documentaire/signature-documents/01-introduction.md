## Objectif de cette section

Cette documentation vous guide dans l'utilisation du module **Signature de documents** de la plateforme Papaours.
Elle permet de comprendre comment gérer les signatures de documents, qu'il s'agisse de signatures manuscrites sur papier ou de signatures électroniques dématérialisées via DocuSeal.

Le module **Signature de documents** permet de :
- Déclarer des demandes de signature sur les documents
- Suivre l'avancement des signatures
- Récupérer les documents signés
- Gérer les visas des organisations

---

## Public visé

Cette documentation s'adresse aux :

- **Gestionnaires administratifs**
- **Chargé(e)s de formation** et coordinateurs pédagogiques
- **Responsables CFA** pour la gestion des visas
- Utilisateurs disposant des **droits d'accès sur les signatures** dans l'application

---

## Permissions nécessaires

Pour utiliser les fonctionnalités du module Signature de documents, l'utilisateur doit disposer des permissions suivantes :

- **Lecture** : consulter les signatures et leur état
- **Écriture** : créer, envoyer et valider des signatures
- **Paramétrage visa** : créer et modifier les visas des organisations
