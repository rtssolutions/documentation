# Annulation d'une inscription

## Vue d'ensemble

L'**annulation d'inscription** est une action administrative qui permet d'annuler complètement un dossier de formation. Cette opération est irréversible.

### Situations permettant l'annulation

L'annulation est possible seulement si le dossier est dans l'état suivant :
- Inscrit à la formation

### Étape 1 - Accéder au dossier

1. Cliquez sur **Dossier de formation** dans le menu principal
2. Recherchez le dossier à annuler
3. Cliquez sur le dossier pour l'ouvrir
5. Si l'inscription peut être annulée, le bouton **Annuler inscription** est visible dans l'encart "Inscription à la formation"

### Étape 2 - Lancer l'annulation

1. Dans le dossier, cliquez sur le bouton **Annuler inscription**
2. Sélectionnez **Annuler l'inscription**
3. Un message de confirmation apparaît
4. Le dossier passe immédiatement à l'état **Inscription annulée**.
5. Le nombre de places disponible dans la session est décrémenté automatiquement.

5. **Attention** : Cette action est définitive
